# Translating Chat App
The Translating Chat App is a web application that allows users to communicate with each other in different languages using automatic translation. The app is built using React, Node.js, and the Libre translate.

# Installation
1. Clone this repository to your local machine.

2. Install the required dependencies by running npm install in both the client and server directories.
  ```shell
    cd client && npm install
  ```
  ```shell
    cd server && npm install
  ```
3. Install Libre Translate and Language Models.
   https://github.com/LibreTranslate/LibreTranslate
   
4. Start the development server by running npm start in both the client, server directories.
5. Start libretranslate server (change port if needed)
  ```shell
  libretranslate --port 5001
  ```
